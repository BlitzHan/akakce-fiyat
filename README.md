# Akakçe Fiyat Takibi

Ürün fiyatlarını Akakçe üzerinden takip etmek için kullanılan basit bir web arayüzü.

## Özellikler

- Akakçe ürün URL'si girerek fiyat bilgisi çekme
- Fiyat geçmişini görsel grafik ile takip etme
- Türkçe arayüz

## Kullanım

Canlı siteye ulaşmak için: **[blitzhan.github.io/akakce-fiyat](https://blitzhan.github.io/akakce-fiyat/)**

Ya da `index.html` dosyasını doğrudan tarayıcınızda açabilirsiniz.

## Teknolojiler

- Vanilla HTML / CSS / JavaScript
- GitHub Pages ile barındırılıyor
